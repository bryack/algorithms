# Правила работы с git (legacy)

## Запрещено самостоятельно выполнять git commit

**Правило:** При запросе "Создай комментарий для git" я должен ТОЛЬКО создать текст комментария и показать команды, но НЕ выполнять `git commit` самостоятельно.

**Что делать:**
1. Создать сообщение коммита на английском языке
2. Показать команды `git add` и `git commit -m "..."`
3. Спросить: "Выполнить коммит или сделаешь сама?"

**Что НЕ делать:**
- ❌ Не выполнять `git commit` без явного разрешения
- ❌ Не делать `git push`
- ❌ Не изменять историю git (rebase, reset и т.д.)

**Исключение:** Если пользователь явно сказал "сделай коммит" или "выполни коммит" — тогда можно выполнить.
