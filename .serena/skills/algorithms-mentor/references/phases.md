# Фазы обучения — подробное описание

## Фаза 1: Массивы + Рекурсия (Недели 3-7)

**Цель**: Освоить 80% паттернов собеседований.

### Блок 1. Hash Maps (Недели 3-4)
- Frequency counter
- Two Sum, Group Anagrams
- Contains Duplicate, Valid Anagram

### Блок 2. Two Pointers & Sliding Window (Недели 5-6)
- Valid Palindrome
- 3Sum
- Longest Substring Without Repeating Characters

### Блок 3. Рекурсия (Неделя 7)
- Факториал, числа Фибоначчи
- Рекурсивный бинарный поиск
- Разворот строки рекурсивно

**Time-constrained practice**: С недели 4 — 2 задачи под таймер (20 мин) еженедельно.

---

## Фаза 2: Линейные структуры (Недели 8-12)

**Цель**: Работа с указателями и индуктивное мышление.

### Блок 1. Стеки и очереди (Недели 8-9)
- Valid Parentheses
- Daily Temperatures
- Min Stack

### Блок 2. Связные списки (Недели 10-11)
- Reverse Linked List
- Merge Two Sorted Lists
- Cycle Detection (Floyd's)

### Блок 3. Divide & Conquer (Неделя 12)
- Merge Sort
- Quick Sort
- Итеративный бинарный поиск

---

## Фаза 3: Деревья и Графы (Недели 13-18)

**Цель**: Иерархическая индукция и обходы.

### Блок 1. Деревья (Недели 13-15)
- Maximum Depth of Binary Tree
- Validate BST
- Lowest Common Ancestor
- Обходы: pre/in/post order, level order

### Блок 2. Графы (Недели 16-18)
- Number of Islands
- Course Schedule
- Word Ladder
- BFS vs DFS

## Фаза 4: Динамическое программирование (Недели 19-20)

**Цель**: Базовые паттерны ДП для джунов.

- Fibonacci (memoization vs tabulation)
- Climbing Stairs
- House Robber

**НЕ углубляться в**: LCS, Edit Distance, 0/1 Knapsack (это middle+)

## Фаза 5: Полировка (Недели 21-24)

**Цель**: Мышление под давлением, реальные собесы.

- Mock interviews (устное решение)
- LeetCode "Top Interview 150" (дожим оставшееся)
- Разбор задач из открытых источников (Яндекс, Тинькофф)

---

## Фаза 6: Подготовка к собеседованию по карточкам

**Цель**: Научиться естественно отвечать на вопросы по Go и архитектуре.

**Расписание**: Ежедневно по 2 карточки из файла «Программирование Карточки.csv»

**Методика переработки одной карточки (6 шагов)**:
1. Прочитать текущий ответ
2. Выделить ключевую идею (1 простое предложение)
3. Переписать от первого лица («Я использую…»)
4. Добавить пример из своего pet-проекта
5. Убрать «книжные» конструкции, добавить связки
6. Проверить: можно ли сказать это на одном дыхании?

**Review**: Раз в неделю — прогон всех переработанных карточек вслух.
