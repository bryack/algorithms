# Workflow: Управление сессиями

## Файлы

- `.session_start` — timestamps начала сессий и перерывов
- `journal.csv` — журнал решённых задач с временем

---

## Команды

### Начало сессии

**Триггер**: "Начинаем новую сессию", "Старт сессии", "Начинаем заниматься", "Сегодня [дата]"

```bash
date "+%Y-%m-%d %H:%M:%S  # SESSION_START" > .session_start
cat .session_start
```

**Ответ:** "✅ Сессия зафиксирована: [время]"

### Перерыв

**Начало перерыва** (триггер: "сейчас перерыв", "перерыв", "отдых"):
```bash
echo "$(date '+%Y-%m-%d %H:%M:%S')  # BREAK_START" >> .session_start
```

**Ответ:** "✅ Перерыв с [время]"

**Окончание перерыва** (триггер: "перерыв окончен", "вернулась", "продолжаем"):
```bash
echo "$(date '+%Y-%m-%d %H:%M:%S')  # BREAK_END" >> .session_start
```

**Ответ:** "✅ Перерыв окончен в [время]. Длительность: [N] минут."

### Запись в journal.csv

**Триггер**: "Запиши в журнал", "Я завершила задачу", "Занеси в журнал"

1. Зафиксировать время:
```bash
date "+%Y-%m-%d %H:%M:%S"
echo "$(date '+%Y-%m-%d %H:%M:%S')  # JOURNAL: [название задачи]" >> .session_start
```

2. Спросить пользователя о времени задачи или рассчитать из `.session_start`

3. Формат записи в `journal.csv`:
```csv
[номер],[дата],[название задачи],[время в минутах],[статус]
```

**Пример:**
```
8,2026-03-23,LeetCode 405: Hexadecimal,90,done
```

### Подсчёт времени

**Важно**: НЕ учитывать время перерывов (BREAK_START → BREAK_END)

**Формула:**
```
Время задачи = (Время окончания - Время начала) - Сумма всех перерывов
```

## Структура .session_start

```
2026-04-04 15:41:35  # SESSION_START
2026-04-04 16:30:00  # BREAK_START
2026-04-04 16:45:00  # BREAK_END
2026-04-04 18:30:00  # JOURNAL: Калькулятор дробей завершение
```

## Пример диалога

```
Пользователь: Начинаем новую сессию
Я: [выполняю date > .session_start]
   ✅ Сессия зафиксирована: 2026-04-04 15:41:35

[проходит время...]

Пользователь: Сейчас перерыв
Я: ✅ Перерыв с 16:30

[проходит время...]

Пользователь: Перерыв окончен
Я: ✅ Перерыв окончен в 16:45. Длительность: 15 минут.

Пользователь: Я завершила задачу
Я: Отлично! Сколько времени заняло? (или: Занесу в журнал как 2 часа?)
```

## Быстрый reference

| Ситуация | Команда |
|----------|---------|
| Начало сессии | `date "+%Y-%m-%d %H:%M:%S # SESSION_START" > .session_start` |
| Перерыв | `echo "$(date '+%Y-%m-%d %H:%M:%S') # BREAK_START" >> .session_start` |
| Конец перерыва | `echo "$(date '+%Y-%m-%d %H:%M:%S') # BREAK_END" >> .session_start` |
| Журнал | `echo "$(date '+%Y-%m-%d %H:%M:%S') # JOURNAL: [название]" >> .session_start` |
